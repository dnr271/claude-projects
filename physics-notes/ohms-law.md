# Ohm's Law Formulas

## Basic Relationship

$$V = I \times R$$

Where:
- **V** = Voltage (Volts, V)
- **I** = Current (Amperes, A)
- **R** = Resistance (Ohms, &Omega;)

## Derived Forms

| Solve for | Formula |
|-----------|---------|
| Voltage | $V = I \times R$ |
| Current | $I = \dfrac{V}{R}$ |
| Resistance | $R = \dfrac{V}{I}$ |

## Power Relations (Ohm's Law + Power)

$$P = V \times I$$

| Solve for | Formula |
|-----------|---------|
| Power | $P = V \times I$ |
| Power | $P = I^2 \times R$ |
| Power | $P = \dfrac{V^2}{R}$ |
| Voltage | $V = \dfrac{P}{I}$ |
| Current | $I = \dfrac{P}{V}$ |
| Resistance | $R = \dfrac{P}{I^2}$ |
| Resistance | $R = \dfrac{V^2}{P}$ |
